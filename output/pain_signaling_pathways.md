## TRPV1-CGRP Nociceptor Axis

**Description**: Nociceptor terminal activation and neuropeptide release axis. TRPV1/TRPA1 detect noxious stimuli -> CGRP/Substance P release -> neurogenic inflammation and central sensitization.

**Direction**: TRPV1/TRPA1 activation -> Ca2+ influx -> CGRP/Substance P release -> NGF/BDNF upregulation -> central sensitization

**Key Genes**: TRPV1, TRPA1, CALCA, CALCB, TAC1, TACR1, NGF, NTRK1, BDNF, NTRK2

**Literature**: PMID: 31636456, PMID: 29487171, PMID: 25303979

**Significance**: Core pain transduction pathway; primary target for peripheral analgesics

## Nav1.7 Action Potential Axis

**Description**: Voltage-gated sodium channel axis for action potential generation and propagation in nociceptors. Nav1.7 acts as threshold channel; Nav1.8 drives the upstroke.

**Direction**: Nav1.7 activation -> threshold depolarization -> Nav1.8-mediated action potential -> pain signal propagation

**Key Genes**: SCN9A, SCN10A, SCN11A, SCN1A, SCN2A, SCN8A

**Literature**: PMID: 30651630, PMID: 25188265, PMID: 17167479

**Significance**: Human genetics support: SCN9A gain-of-function = erythromelalgia; loss-of-function = congenital insensitivity to pain

**RA-PainKG Status**: Found: SCN9A[KG,TrackA,TrackB], SCN10A[KG], SCN11A[KG], SCN1A[KG,TrackB], SCN2A[KG,TrackB], SCN8A[KG,TrackA,TrackB]

## TNFa-NGF Inflammatory Pain Axis

**Description**: Inflammatory cytokine-driven peripheral sensitization. TNF-a and IL-1b stimulate NGF production -> TrkA activation -> TRPV1 sensitization.

**Direction**: TNF-a/IL-1b -> NGF upregulation -> TrkA -> TRPV1 phosphorylation/sensitization -> hyperalgesia

**Key Genes**: TNF, TNFRSF1A, NGF, NTRK1, NGFR, IL1B, IL6, PTGS2

**Literature**: PMID: 20393189, PMID: 21788404, PMID: 29230043

**Significance**: Links immune activation to pain hypersensitivity; explains pain despite inflammation control

## Opioid Endogenous Analgesia Axis

**Description**: Endogenous opioid system for descending pain modulation. mu/delta/kappa receptors mediate analgesia at spinal and supraspinal levels.

**Direction**: POMC -> b-endorphin / PENK -> enkephalins / PDYN -> dynorphins -> mu/delta/kappa receptors -> descending pain inhibition

**Key Genes**: OPRM1, OPRD1, OPRK1, POMC, PENK, PDYN

**Literature**: PMID: 28624224, PMID: 25743836

**Significance**: Key perioperative target; mu receptor is the primary target of clinical opioids

**RA-PainKG Status**: Found: OPRM1[KG], OPRD1[KG], OPRK1[KG], POMC[KG,TrackA,TrackB], PENK[KG,TrackA,TrackB], PDYN[KG]

## GABAergic Descending Inhibition

**Description**: GABA/glycine-mediated inhibitory neurotransmission in spinal dorsal horn. Loss of inhibition contributes to chronic pain.

**Direction**: GABA release -> GABAA receptor -> Cl- influx -> hyperpolarization -> inhibition of pain transmission

**Key Genes**: GABRA1, GABRA2, GABRB2, GABRG2, GABRB3, GLRA1, GLRB

**Literature**: PMID: 25855173, PMID: 28492257

**Significance**: Anesthetic targets; benzodiazepines and propofol act via GABA-A receptors
