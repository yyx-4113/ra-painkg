# Significance Statement

This work establishes that single-split knowledge graph (KG) evaluation—the prevailing practice in perturbation prediction benchmarking—produces misleading conclusions. Through a multi-split ablation design with causal controls (degree-preserving randomization, domain-centric reduction), we demonstrate that graph connectivity, not edge semantics, drives prediction performance. The open-source benchmark framework with paired statistics, Kendall's W ranking consistency, and sensitivity analyses provides a methodological standard for future KG evaluation. These findings apply broadly to biomedical informatics applications where domain-specific prior knowledge is integrated into machine learning pipelines, from drug target prioritization to disease gene discovery.
